# 🎬 Video Extractor

多浏览器兼容的浏览器扩展，用于检测和提取网页中的视频源，支持预览和下载。

## 功能

- **双通道检测**：DOM 扫描（`<video>` 标签、播放器配置）+ 网络请求监控
- **🎯 广泛覆盖**：支持 MP4、M3U8、TS、WebM、MKV、FLV 等格式
- **👁️ 视频预览**：点击即可在弹窗中预览视频
- **⬇ 一键下载**：通过浏览器下载管理器保存视频
- **📋 复制地址**：快速复制视频 URL 到剪贴板
- **🏷️ 智能识别**：自动识别 jwplayer、videojs、DPlayer 等常见播放器

## 安装

### 第一步：下载代码

点击本页面右上角的绿色 **「<> Code」** 按钮 → **「Download ZIP」**，下载后解压到任意文件夹。

---

### 第二步：加载扩展

#### Edge 浏览器（推荐）

1. 打开 Edge，在地址栏输入 `edge://extensions` 并回车
2. 打开左下角的 **「开发人员模式」** 开关
3. 点击上方出现的 **「加载解压缩的扩展」** 按钮
4. 在弹出的文件夹选择框中，选择刚才解压的 `video-extractor-main` 文件夹
5. 完成！工具栏右侧会出现 🎬 图标

#### Chrome 浏览器

1. 打开 Chrome，在地址栏输入 `chrome://extensions` 并回车
2. 打开右上角的 **「开发者模式」** 开关
3. 点击左上角 **「加载已解压的扩展程序」** 按钮
4. 选择解压后的 `video-extractor-main` 文件夹
5. 完成！工具栏右侧会出现 🎬 图标

#### Firefox 浏览器

1. 打开 Firefox，在地址栏输入 `about:debugging#/runtime/this-firefox` 并回车
2. 点击 **「临时载入附加组件…」** 按钮
3. 选择解压文件夹中的 `manifest.json` 文件
4. 完成！

> 💡 **注意：** 开发者模式加载的扩展每次打开浏览器可能会有「请停用开发者模式扩展」的提示，直接点 × 关闭即可，不影响使用。

## 使用

1. 打开任意包含视频的网页
2. 点击浏览器工具栏右侧的 🎬 扩展图标
3. **「页面视频」标签** — 自动显示页面中检测到的视频（`<video>` 标签、播放器等）
4. **「网络捕获」标签** — 实时监控页面加载的视频网络请求
5. 点击任意视频条目可展开操作：
   - **👁️ 预览** — 在弹窗中直接播放视频
   - **⬇ 下载** — 通过浏览器下载管理器保存文件
   - **📋 复制** — 一键复制视频地址到剪贴板

> 💡 **提示：** 有些网站的视频是动态加载的，切换到「网络捕获」标签后，刷新页面可以看到更多视频源。

## 常见问题

### 为什么检测不到视频？
- 某些网站在你打开扩展后才加载视频，尝试**刷新页面**后再查看
- M3U8 等流媒体格式只出现在「网络捕获」标签中，需要刷新页面才能捕获
- 部分网站使用 DRM 保护（如 Netflix、腾讯视频 VIP），无法获取视频地址

### Edge 提示「请停用开发者模式扩展」怎么办？
这是 Edge 的安全提示，点 × 关闭即可，不影响扩展功能。

### 支持哪些视频格式？
MP4、M3U8、TS、WebM、MKV、FLV、AVI、MOV、OGG、WMV、M4V、F4V、3GP

### 扩展会收集我的数据吗？
不会。所有检测数据仅存储在本地浏览器，不会上传到任何服务器。代码完全开源，你可以自行审查。

## 项目结构

```
video-extractor/
├── manifest.json            # MV3 清单（Chrome/Firefox/Edge 兼容）
├── background/
│   └── background.js        # Service Worker：网络请求监控
├── content/
│   └── content.js           # Content Script：DOM 视频扫描
├── popup/
│   ├── popup.html           # 弹出窗口
│   ├── popup.css            # 样式
│   └── popup.js             # 交互逻辑
├── icons/                   # 扩展图标
└── generate-icons.js        # 图标生成脚本
```
